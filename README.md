My site. Uses:

* Mustache templating
* LESS compiling

## Getting started

Dependencies:
* node && npm
* gulp-cli (global) && gulp (local)

```
git clone https://github.com/robertdenton/gulp-starter.git
cd project/src
npm install

# Start dev server
gulp serve

# Build static site
gulp
```

Also, shout out to [simpleicons.org](https://simpleicons.org/) for the share icons.
